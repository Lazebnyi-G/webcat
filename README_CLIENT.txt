Как открыть сайт

1. Распакуйте архив в любую папку.
2. Запустите файл START_SITE.bat двойным кликом.
3. Сайт откроется в браузере по умолчанию.

Если скрипт не запускается, откройте файл index.html вручную.

Админ-зона

- Ссылка: Админ в верхнем меню или адрес index.html#admin
- Код входа: 2468

Важно

- Это статический сайт без сервера.
- Изменения в админ-зоне сохраняются только в браузере, где они были сделаны.
- Для публикации в интернете нужен статический хостинг.
