# Content Maker Portfolio

Статический сайт-портфолио для контент-мейкера. В проекте есть публичная страница, видео на главном экране и простая клиентская админ-зона.

## Главное

- `index.html` - основная страница сайта.
- `assets/` - изображения, видео и загрузочная анимация.
- `docs/` - документация проекта.
- `variants/` - отдельные экспериментальные версии страницы.
- `dev/` - локальные файлы разработки: логи и профиль браузера.

Подробная карта файлов: `docs/FILE_MAP.md`.

## Возможности

- Главный экран с крупным именем и видео-превью.
- Разделы: работы, направления, процесс, о себе, контакты.
- Адаптивная верстка для desktop и mobile.
- Анимированный canvas-фон с частицами.
- Админ-зона по адресу `#admin`.
- Редактирование работ, текстов, контактов, изображений и видео через браузер.
- Сохранение данных в `localStorage` конкретного браузера.

## Как открыть

1. Откройте `index.html` в браузере.
2. Для админ-зоны перейдите по ссылке `Админ` или добавьте `#admin` к адресу страницы.
3. Код входа по умолчанию: `2468`.

## Важные ограничения

- Это статический прототип без серверной части.
- Данные админки сохраняются только в браузере пользователя.
- Код входа находится в исходном коде, поэтому это не полноценная защита.
- Для продакшена нужна серверная админка, база данных и нормальная авторизация.

## Передача заказчику

Проект можно передавать как git-репозиторий или архив папки. Для просмотра достаточно открыть `index.html`.

Если нужна публикация в интернете, проект можно разместить на статическом хостинге: GitHub Pages, Netlify, Vercel или обычном хостинге с поддержкой HTML-файлов.
